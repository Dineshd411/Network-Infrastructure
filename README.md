## What's in This Repository

This portfolio documents structured, hands-on IT and networking training — moving from fundamentals through switching/routing, structured cabling, server administration, and firewall configuration. Labs are separated by whether they were done in **Cisco Packet Tracer** or on **real hardware**, since the failure modes genuinely differ between the two.

- Networking fundamentals — OSI & TCP/IP models, IPv4 addressing, Subnetting, VLSM, CIDR, DNS, DHCP, NAT
- Cisco switching and routing — VLANs, Inter-VLAN Routing (Router-on-a-Stick & SVI), STP, EtherChannel/Port Channel, Static & Dynamic Routing (RIP, OSPF)
- Access Control Lists — Standard/Extended ACLs, wildcard masks, Inter-VLAN ACL policies on SVIs
- Structured cabling — Cat6 copper cabling, RJ45 crimping, keystone termination, patch panels, MDF/IDF, fiber basics (SMF/MMF, LC/SC)
- Server & virtualization — Active Directory, Domain Controller, OU/User/Group management, Group Policy, VMware Workstation, Oracle VirtualBox
- Next-generation firewalls — FortiGate (FortiOS) and Endian UTM, interface/NAT/policy configuration, zone segmentation (Green/Red), web/URL filtering
- Wireless networking — SSID/multiple SSID design, Cisco Access Points, SOHO router configuration, guest/corporate VLAN separation
- Network security & monitoring — Wireshark, Nmap, NetDiscover, Nessus, Splunk (SIEM), VirusTotal, Cisco Talos, MXToolbox
